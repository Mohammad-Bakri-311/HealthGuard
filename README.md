# HealthGuard - Healthcare Management System

HealthGuard is a healthcare system built using PHP and MySQL.

## Status
Work in Progress

## Features

### Admin
- Manage doctors
- Manage patients
- Manage users
- Manage illnesses
- Manage drugs

### Doctor
- View patients
- Manage appointments
- View reports

### Patient
- Book appointments
- Find doctors
- View reports

## Technologies
- PHP
- MySQL
- HTML
- CSS

## How to Run

1. Put the project in:
C:\wamp64\www\HealthGuard

2. Start WAMP

3. Open:
http://localhost/HealthGuard/
